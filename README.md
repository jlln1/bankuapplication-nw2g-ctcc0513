# bankuapplication-nw2g-ctcc0513
            This is a Java banking system, and the name of the system is Bank-U. 
            It is pronounced as "Banku," which has a similar sound to the Filipino term for bank, and the "U" in the system's name indicates that the system is for the users. 
            Bank-U is a desktop banking application, the GUI for this system is developed using Java Swing and integrates all the fundamental components 
            like JPanel, JLabel, JButton, and JTextArea to develop an interface that is visually appealing and user-friendly 
            and provide users with an intuitive and secure environment to manage basic banking operations. 
            Additionally, the system consists of two primary screens, the Login Screen ensures security access, 
            while the Dashboard Screen allows smooth navigation to main banking functionalities. 
            The system will provide options to check balances, deposit or withdraw funds, transfer money to other accounts, view histories of transactions, 
            and generate receipts for recent transactions.

            Moreover, to manage and store the transaction history efficiently, Bank-U uses basic data structures such as LinkedList and Queue. 
            The LinkedList enables active transaction log management, with efficient adding and retrieving of records. 
            On the other hand, the Queue manages a receipt system with a certain capacity so that the storage limits will not be exceeded. 
            Also, Bank-U delivers detailed transaction tracking, giving users a full view of their financial activities. 
            However, despite its strong functionality, Bank-U is limited to simulated banking operations, as it does not integrate with external banking systems 
            and it still not connected to a database or has a support advanced security measures like encryption.

            Hence, Bank-U is a dependable desktop banking application that combines Java Swing components for its graphical interface 
            with LinkedList and Queue data structures for effective data management. In addition, it gives a clear, user-friendly, and secure platform to meet 
            basic banking needs while being an outstanding solution for desktop users and highlighting potential opportunities for future enhancements
